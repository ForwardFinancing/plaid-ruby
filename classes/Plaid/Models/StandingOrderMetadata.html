Class Plaid::Models::StandingOrderMetadata < Plaid::Models::BaseModel

  • lib/plaid/models.rb

A representation of standing order metadata for an institution.

Attributes
[R] supports_standing_order_end_date

The Boolean flag indicating if the institution supports end date for standing orders.
[R] supports_standing_order_negative_execution_days

The Boolean flag indicating if the institution supports negative execution days for standing orders.
[R] valid_standing_order_intervals

The Array of valid standing order intervals for this institution. E.g. [“WEEKLY”, “MONTHLY”].
