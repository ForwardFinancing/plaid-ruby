Class Plaid::Models::InstitutionPaymentInitiationMetadata < Plaid::Models::BaseModel

A representation of an institution's payment initiation metadata.

Attributes
[R] maximum_payment_amount

The map of maximum payment amount per currency for this institution. E.g. {“GBP”=>“1000000”}.
[R] standing_order_metadata

The standing order metadata for this institution.
[R] supports_international_payments

The Boolean flag indicating if the institution supports international payments.

The Boolean flag indicating if the institution supports refund details.
